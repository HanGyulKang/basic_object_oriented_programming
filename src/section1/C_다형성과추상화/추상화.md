* 데이터나 프로세스 등을 의미가 비슷한 개념이나 의미있는 표현으로 정의하는 과정
  * 즉, 개념화 또는 표현을 도출하는 것
* 두 가지 방식의 추상화
  * 특정한 성질
  * 공통 성질(일반화)

---

* 간단한 예
  * DB의 USER 테이블 : 아이디, 이름, 이메일
  * Money 클래스 : 통화, 금액
  * 프린터 : HP MXXX, 삼성 SL-M2XXX
  * GPU : 지포스, 라데온

---
### 타입 추상화
* 여러 구현 클래스를 대표하는 상위 타입 도출
  * 보편적으로 __Interface type__ 으로 추상화 한다.
  * 추상화 타입과 구현은 __타입 상속__ 으로 연결한다.
